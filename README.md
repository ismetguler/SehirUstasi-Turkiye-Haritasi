## About

**Şehir Ustası** is a published mobile game that turns learning Turkish geography into a challenge. Players locate and guess cities on an interactive map of Turkey and compete to become the "city master." The app is **live on the Google Play Store with 900+ installs**.

## Features

- 🗺️ Interactive map of Turkey with all 81 provinces
- 🎯 Geography-based quiz & guessing mechanics
- 🏆 Score tracking and progression
- 📈 AdMob monetization integrated
- ☁️ Firebase backend for data and analytics

## Tech stack

- **Kotlin** (Android)
- **Firebase** (Firestore / Analytics)
- **Google AdMob**
